This is a [Next.js](https://nextjs.org) project for a restrained product-style landing page.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Style Guide Route

The project includes a dedicated route: [http://localhost:3000/style-guide](http://localhost:3000/style-guide).

This page is an internal working tool that fixes:

- visual direction,
- design tokens,
- typography scale,
- base reusable components for future landing sections.

Main visual system documentation is in `docs/style-guide.md`.

## Stack

- Next.js 16 (App Router)
- TypeScript
- CSS variables-based minimal visual system
